# Images

This folder contains visualizations generated during the machine learning workflow and model evaluation process.

## Files

### churn_distribution.png

Shows the distribution of churned and retained customers in the dataset.

### feature_importance.png

Displays the most influential features affecting customer churn prediction.

### model_comparison.png

Compares the performance of different machine learning algorithms evaluated during model selection.

Models compared:

* Logistic Regression
* Decision Tree
* Random Forest
* Gradient Boosting

### confusion_matrix.png

Visual representation of prediction results showing:

* True Positives
* True Negatives
* False Positives
* False Negatives

### roc_curve.png

Receiver Operating Characteristic (ROC) Curve for the final Logistic Regression model.

**ROC-AUC Score:** 0.836

The ROC curve demonstrates the model's ability to distinguish between churn and non-churn customers.

## Purpose

These visualizations support exploratory data analysis, feature selection, model comparison, and performance evaluation. They provide graphical evidence of the effectiveness of the customer churn prediction model.
